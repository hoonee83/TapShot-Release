TapShot v1.9.0

전체 캡처, 영역 캡처, 3초 뒤 영역 캡처, 마지막 영역 반복, 스티치를 지원하는 Windows 전용 캡처 도구입니다. 캡처 결과는 미리보기 탭으로 열리고, 자동 저장과 클립보드 복사가 함께 처리됩니다.

최신 릴리즈 보기 스티치 기능

주요 기능

v1.9.0은 캡처 흐름, 스티치, 지연 캡처, 업데이트 확인을 갖춘 안정 버전입니다.

전체 캡처

현재 가상 화면 전체를 캡처합니다. 다중 모니터 환경도 지원합니다.

영역 캡처

마우스 드래그로 영역을 선택하고, 키보드로 위치를 보정할 수 있습니다.

3초 뒤 캡처

컨텍스트 메뉴나 팝업처럼 바로 사라지는 화면을 캡처하기 좋습니다.

마지막 반복

직전에 선택한 영역을 다시 캡처합니다. 동일 영역 반복 기록에 적합합니다.

주석 도구

펜, 선, 화살표, 사각형, 원형 도구로 캡처 이미지 위에 표시를 남깁니다.

단축키 설정

전체 캡처, 영역 캡처, 3초 뒤 캡처, 마지막 반복 단축키를 바꿀 수 있습니다.

자동 저장

캡처 및 스티치 결과는 PNG로 저장되고 클립보드에도 복사됩니다.

업데이트 확인

GitHub Releases 최신 버전 정보를 확인하고 다운로드 페이지로 이동합니다.

스티치

열려 있는 미리보기 탭 여러 개를 세로로 합쳐 하나의 이미지로 만듭니다.

  1. 합칠 캡처 이미지가 들어 있는 미리보기 탭을 준비합니다.
  2. 탭을 드래그해서 왼쪽부터 오른쪽 순서로 정렬합니다.
  3. 스티치 다이얼로그에서 합칠 탭을 체크합니다.
  4. 폭 맞춤 옵션을 선택하고 실행합니다.
폭 옵션 동작
원본 크기 유지 각 이미지를 원본 크기로 유지하고 왼쪽 정렬로 합칩니다.
가장 좁은 폭에 맞춤 기준보다 큰 이미지만 축소합니다. 기본값입니다.
가장 넓은 폭에 맞춤 모든 이미지를 가장 넓은 폭에 맞춰 확대 또는 축소합니다.

설치와 업데이트

TapShot은 GitHub Releases를 기준으로 배포하고, 앱 안에서 최신 릴리즈 정보를 확인할 수 있습니다.

최신 다운로드 릴리즈 페이지에서 최신 TapShot_vX.Y.Z_Setup.exe 설치파일을 받을 수 있습니다. https://github.com/hoonee83/TapShot-Release/releases/latest
설치 파일 TapShot_v1.9.0_Setup.exe
설치 후 실행 파일 TapShot.exe
자동 실행 옵션 설치 중 또는 앱 설정에서 Windows 시작 시 자동 실행을 선택할 수 있습니다.
